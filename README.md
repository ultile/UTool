# UTool
[![License: GPL v3](https://img.shields.io/badge/License-GPL_v3-blue.svg)](https://github.com/ultile/UTool/blob/main/LICENSE)

**A Text user Interface project I've done to pass time. Installs and Uninstalls OpenAsar for all builds of Discord on Windows 10/11**

# Features
Install OpenAsar
Uninstall OpenAsar

# Compilation Command Used
gcc -Wall -Werror -Wpedantic -fdiagnostics-color=always -g main.c -o utile.exe -lurlmon
